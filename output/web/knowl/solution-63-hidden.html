As we did before, we first need to define what year will correspond to \(n = 0\text{.}\) Since we know the population in 2008, it would make sense to have 2008 correspond to \(n = 0\text{,}\) so \(P_0 = 245000\text{.}\) The year 2014 would then be \(n = 6\text{.}\)
We know the growth rate is 3%, giving \(r = 0.03\text{.}\)
Using the explicit form:
\(P_6 = \left( 1+0.03 \right)^6 (245,000) = 1.19405(245,000) = 292,542.25\)
The model predicts that in 2014, Olympia would have a population of about 293 thousand people.
