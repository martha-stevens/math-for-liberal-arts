In order to compute the probability, we need to count the total number of ways six numbers can be drawn, and the number of ways the six numbers on the player’s ticket could match the six numbers drawn from the machine. Since there is no stipulation that the numbers be in any particular order, the number of possible outcomes of the lottery drawing is \(_{48}C_{6}=12,271,512\text{.}\) Of these possible outcomes, only one would match all six numbers on the player’s ticket, so the probability of winning the grand prize is:
\begin{equation*} \frac{_{6}C_{6}}{_{48}C_{6}}=\frac{1}{12271512}\approx 0.0000000815 \end{equation*}
