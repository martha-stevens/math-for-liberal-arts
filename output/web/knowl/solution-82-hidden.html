In this example, we’re looking for \(P_0\text{.}\)
\(r = 0.04\)
\(k = 4\) (4 quarters in 1 year)
\(N = 18\)
\(P_{18} = \$40,000\)
In this case, we’re going to have to set up the equation, and solve for \(P_0\text{.}\)
\begin{align*} 40000 \amp =P_0 \left( 1 + \frac{0.04}{4}\right)^{4 \cdot 18}\\ 4000 \amp = P_0 \left( 2.0471 \right)\\ P_0 \amp = \frac{40000}{2.0471}=19539.84 \end{align*}
So you would need to deposit $19,539.84 now to have $40,000 in 18 years.
