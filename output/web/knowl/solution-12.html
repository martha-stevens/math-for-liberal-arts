Solution 1.2.4.1.

We can set up a proportion by setting equal two \(\frac{\text{map inches}}{\text{real miles}}\) rates, and introducing a variable, \(x\text{,}\) to represent the unknown quantity - the mile distance between the cities.
\begin{equation*} \frac{\frac{1}{2} \text{ inch}}{3 \text{ miles}}=\frac{2 \frac{1}{4} \text{ inch}}{x \text{ miles}} \end{equation*}
Then we can set the cross products equal to solve:
\begin{equation*} \frac{1}{2}x = \left(2 \frac{1}{4} \right)\cdot 3 \end{equation*}
\begin{equation*} \frac{1}{2}x = \frac{9}{4} \cdot \frac{3}{1} \end{equation*}
\begin{equation*} \frac{1}{2}x = \frac{27}{4} \end{equation*}
Finally, we can divide both sides by \(\frac{1}{2}\) to solve for \(x\text{.}\) Recall, dividing by a fraction is the same as multiplying by its reciprocal. For example,
\begin{equation*} \frac{27}{4} \div \frac{1}{2} = \frac{27}{4}\cdot \frac{2}{1} \end{equation*}
To solve for x, when can divide both sides by \(\frac{1}{2}\text{,}\) which is equivalent to multiplying both sides by 2.
\begin{equation*} \frac{2}{1}\cdot \frac{1}{2}x = \frac{27}{4} \cdot \frac{2}{1} \end{equation*}
\begin{equation*} x = \frac{54}{4} = \frac{27}{2} = 13 \frac{1}{2} \text{ miles} \end{equation*}
