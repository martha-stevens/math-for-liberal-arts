Definition 2.2.1. Union, Intersection, Complement.

The union of two sets contains all the elements contained in either set (or both sets). The union is notated \(A \cup B\text{.}\) More formally, \(x \in A \cup B \) if \(x \in A\) or \(x \in B\) (or both).
The intersection of two sets contains only the elements that are in both sets. The intersection is notated \(A \cap B\text{.}\) More formally, \(x \in A \cap B \text{ if } x \in A \text{ and } x \in B\text{.}\)
The complement of a set A contains everything that is not in the set A. The complement is notated \(A^C\) or \(A'\text{.}\)
