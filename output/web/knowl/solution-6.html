Solution 1.1.10.1.

To answer this question, suppose the value started at $100. After one week, the value dropped by 60%:
\begin{equation*} \$100 - \$100 \cdot 0.60 = \$100 - \$60 = \$40 \end{equation*}
In the next week, notice that base of the percent has changed to the new value, $40.
Computing the 75% increase:
\begin{equation*} \$40 + \$40 \cdot 0.75 = \$40 + \$30 = \$70 \end{equation*}
In the end, the stock is still $30 lower, or $30 $100 = 30% lower, valued than it started.
