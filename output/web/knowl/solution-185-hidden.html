This one requires that we solve for a missing value rather than for a missing z-score, so we just need to fill in our formula with what we know and solve for the missing value:
\begin{equation*} z = \frac{x - \mu}{\sigma} \end{equation*}
\begin{equation*} 2.403 = \frac{x - 63}{4.25} \end{equation*}
To solve for \(x\text{,}\) we can mutliply both sides by \(4.25\)
\begin{equation*} 10.213 = x - 63 \end{equation*}
\begin{equation*} 73.213 = x \end{equation*}
\(73.213\) has a z-score of \(2.403\text{.}\)
