For the numerator, we need the number of ways to draw one Ace and four other cards (none of them Aces) from the deck. Since there are four Aces and we want exactly one of them, there will be \(_{4}C_{1}\) ways to select one Ace; since there are 48 non-Aces and we want 4 of them, there will be \(_{48}C_{4}\) ways to select the four non-Aces. Now we use the Basic Counting Rule to calculate that there will be \(_{4}C_{1}\cdot _{48}C_{4}\) ways to choose one ace and four non-Aces.
