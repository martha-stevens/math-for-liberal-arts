Solution 12.4.3.1.

First, find the average constituency of each state.
\begin{equation*} A = \frac{5,900,000}{7}=842,857 \end{equation*}
\begin{equation*} B = \frac{3,800,000}{5}=760,000 \end{equation*}
The difference in these values is the absolute unfairness. Absolute unfairness should be a positive number, so take the absolute value of the difference if necessary.
\begin{equation*} \text{AUF}=842,857-760,000 = 82,857 \end{equation*}
\begin{equation*} \text{RUF}=\frac{82,847}{760,000}=0.109 \end{equation*}
The absolute unfairness is \(82,857\) people. The relative unfairness is \(0.109 = 10.9\%\text{.}\)
