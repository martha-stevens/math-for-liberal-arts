Solution 1.1.9.1.

When we make comparisons, we must ask first whether an absolute or relative comparison. The absolute difference is \(215-75=140\text{.}\) From this, we could say “Albertsons has 140 more stores than QFC.” However, if you wrote this in an article or paper, that number does not mean much. The relative difference may be more meaningful. There are two different relative changes we could calculate, depending on which store we use as the base:
Using QFC as the base,
\begin{equation*} \frac{140}{75} = 1.867 = 186.7\% \end{equation*}
This tells us Albertsons is 186.7% larger than QFC.
Using Albertsons as the base,
\begin{equation*} \frac{140}{215}= 0.651 = 65.1\% \end{equation*}
This tells us QFC is 65.1% smaller than Albertsons.
Notice both of these are showing percent differences. We could also calculate the size of Albertsons relative to QFC:
\begin{equation*} \frac{215}{75} = 2.867 \end{equation*}
which tells us Albertsons is 2.867 times the size of QFC. Likewise, we could calculate the size of QFC relative to Albertsons:
\begin{equation*} \frac{75}{215}= 0.349 \end{equation*}
which tells us that QFC is 34.9% of the size of Albertsons.
