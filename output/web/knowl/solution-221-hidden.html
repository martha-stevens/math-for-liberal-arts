The average constituency gives the average number of people each member of Congress represents. We can take the population and divide it by the number of representatives to find the average constituency.
\begin{equation*} A = \frac{5,900,000}{7}=842,857 \end{equation*}
\begin{equation*} B = \frac{3,800,000}{5}=760,000 \end{equation*}
Since the average constituency in state A is larger than state B, state A is more poorly represented. The representatives in state A represent more people than in state B.
